# ESP32PWM 类使用说明

这是一个轻量级、兼容性强的 ESP32 PWM 封装类，基于 LEDC 外设，专为实现平滑的**呼吸灯**和**非阻塞调光**效果而设计。

## ✨ 核心功能

- **硬件驱动**：基于 ESP32 LEDC 外设，不占用 CPU 资源。
- **非阻塞渐变**：内置 `fadeTo()` 动画引擎，无需 `delay()` 即可实现平滑过渡。
- **智能兼容**：自动识别并支持 ESP32 Arduino Core **v2.x** 和 **v3.x** 版本。
- **百分比控制**：支持直接使用 0-100% 设置亮度，无需手动计算占空比数值。

## 🚀 快速上手

### 1. 引入文件

将 `ESP32PWM.h` 和 `ESP32PWM.cpp` 放入你的项目文件夹（src 或者是同级目录）。

### 2. 最小代码示例

```
#include "ESP32PWM.h"

// 定义对象: GPIO 2, 通道 0, 频率 5000Hz, 8位分辨率
ESP32PWM led(2, 0, 5000, 8);

void setup() {
    led.begin(); // 初始化硬件
    
    // 立即开启 (50% 亮度)
    led.setPercent(50);
    
    // 2秒内渐变到 100%
    led.fadeTo(100, 2000);
}

void loop() {
    // 必须在 loop 中不断调用，用于驱动渐变动画
    led.loop();
}
```

## ⚙️ API 参考

### 1. 初始化与配置

| **方法**                         | **说明**                                                     |
| -------------------------------- | ------------------------------------------------------------ |
| `ESP32PWM(pin, ch, freq, res)`   | **构造函数**。 • `pin`: 输出引脚 (GPIO) • `ch`: 通道 (0-15, v3.0+自动管理) • `freq`: 频率 (默认 5000Hz) • `res`: 分辨率 (默认 8bit) |
| `void begin()`                   | **初始化硬件**。必须在 `setup()` 中调用。                    |
| `void setFrequency(double freq)` | **修改频率**。在运行时动态改变 PWM 频率。                    |

### 2. 亮度控制

| **方法**                         | **说明**                                                     |
| -------------------------------- | ------------------------------------------------------------ |
| `void setDuty(uint32_t duty)`    | **设置原始值**。直接设置 PWM 寄存器数值 (如 8bit 为 0-255)。 |
| `void setPercent(float percent)` | **设置百分比**。设置 0.0 - 100.0 的亮度，自动映射到当前分辨率。 |
| `uint32_t getMaxDuty()`          | **获取最大值**。返回当前分辨率下的最大占空比数值 (如 255)。  |

### 3. 动画与渐变

| **方法**                  | **说明**                                                     |
| ------------------------- | ------------------------------------------------------------ |
| `void fadeTo(target, ms)` | **开始渐变**。 • `target`: 目标百分比 (0-100) • `ms`: 持续时间 (毫秒) 此函数**非阻塞**，调用后立即返回。 |
| `void loop()`             | **驱动动画**。必须在主 `loop()` 中持续调用，负责计算和更新渐变数值。 |
| `bool isFading()`         | **查询状态**。返回 `true` 表示动画正在进行，`false` 表示动画结束或静止。 |

## 💡 常见应用场景

### 场景：呼吸灯逻辑

利用 `isFading()` 标志位，可以轻松写出非阻塞的呼吸灯逻辑：

```
void loop() {
    led.loop(); // 驱动动画核心

    // 当一次渐变完成后，自动反转方向
    if (!led.isFading()) {
        static bool dir = true;
        if (dir) led.fadeTo(100, 1000); // 吸气：1秒变亮
        else     led.fadeTo(0, 1000);   // 呼气：1秒变暗
        dir = !dir;
    }
}
```
